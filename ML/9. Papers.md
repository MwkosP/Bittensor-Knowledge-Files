## Reading Tips

- Start at Stage 1 even if you know classical ML — reading the originals builds intuition.
- For each paper note: **what problem it solved**, **the key idea in one sentence**, **which folder it connects to**.
- Stages 1–3 can be done in a few weeks. Stages 4–7 are ongoing.
