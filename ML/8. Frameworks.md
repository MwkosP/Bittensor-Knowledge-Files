## 1. PyTorch

The default framework for deep learning research and production.

### Define a model

```python
import torch.nn as nn

class MLP(nn.Module):
    def __init__(self):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(128, 64),
            nn.ReLU(),
            nn.Linear(64, 10)
        )
    def forward(self, x):
        return self.net(x)
```

### Training loop

```python
optimizer = torch.optim.AdamW(model.parameters(), lr=1e-3)
criterion = nn.CrossEntropyLoss()

for X, y in dataloader:
    optimizer.zero_grad()
    loss = criterion(model(X), y)
    loss.backward()
    optimizer.step()
```

### Inference

```python
model.eval()
with torch.no_grad():
    preds = model(X)
```

### Save / load

```python
torch.save(model.state_dict(), "model.pt")
model.load_state_dict(torch.load("model.pt"))
```

## 2. HuggingFace Transformers

Load and use pretrained models (BERT, GPT, T5, LLaMA, etc.).

### Load model + tokenizer

```python
from transformers import AutoTokenizer, AutoModelForSequenceClassification

tokenizer = AutoTokenizer.from_pretrained("bert-base-uncased")
model = AutoModelForSequenceClassification.from_pretrained("bert-base-uncased")
```

### Inference

```python
inputs = tokenizer("Hello world", return_tensors="pt")
outputs = model(**inputs)
logits = outputs.logits
```

### Fine-tune with Trainer

```python
from transformers import TrainingArguments, Trainer

args = TrainingArguments(output_dir="out", num_train_epochs=3, per_device_train_batch_size=16)
trainer = Trainer(model=model, args=args, train_dataset=train_ds, eval_dataset=val_ds)
trainer.train()
```

### Pipeline (quick inference)

```python
from transformers import pipeline

classifier = pipeline("sentiment-analysis")
classifier("I love this movie")  # [{'label': 'POSITIVE', 'score': 0.99}]
```

## 4. scikit-learn

Classical ML, preprocessing, evaluation.

### Train a model

```python
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import train_test_split

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)
model = RandomForestClassifier(n_estimators=100)
model.fit(X_train, y_train)
model.score(X_test, y_test)
```

### Preprocessing pipeline

```python
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler
from sklearn.linear_model import LogisticRegression

pipe = Pipeline([("scaler", StandardScaler()), ("clf", LogisticRegression())])
pipe.fit(X_train, y_train)
```

### Metrics

```python
from sklearn.metrics import classification_report
print(classification_report(y_test, model.predict(X_test)))
```

## 5. XGBoost / LightGBM

Gradient boosting — best default for tabular data.

```python
import xgboost as xgb

model = xgb.XGBClassifier(n_estimators=300, learning_rate=0.05, max_depth=6)
model.fit(X_train, y_train, eval_set=[(X_val, y_val)], early_stopping_rounds=10)
model.predict(X_test)
```

```python
import lightgbm as lgb

model = lgb.LGBMClassifier(n_estimators=300, learning_rate=0.05)
model.fit(X_train, y_train, eval_set=[(X_val, y_val)], callbacks=[lgb.early_stopping(10)])
```

LightGBM is generally faster than XGBoost on large datasets.

## 6. JAX

NumPy-like API with autodiff and JIT compilation. Used in research and large-scale training (Google).

```python
import jax.numpy as jnp
from jax import grad, jit

def loss(w, x, y):
    return jnp.mean((x @ w - y) ** 2)

grad_fn = jit(grad(loss))   # compiled gradient function
grads = grad_fn(w, X, y)
```

Most HuggingFace models support JAX via `FlaxModel` variants.

## 7. ONNX

Export any trained model to a portable format for deployment.

```python
# Export from PyTorch
torch.onnx.export(model, dummy_input, "model.onnx",
                  input_names=["input"], output_names=["output"],
                  dynamic_axes={"input": {0: "batch_size"}})

# Run with ONNX Runtime
import onnxruntime as ort
sess = ort.InferenceSession("model.onnx")
out = sess.run(None, {"input": X_numpy})
```
