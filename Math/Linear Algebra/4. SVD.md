




# Linear Algebra — SVD (Singular Value Decomposition)

---

## Core Idea

Eigendecomposition only works for square matrices.  
**SVD works for any matrix** — it's the generalization.

Every matrix `A ∈ ℝ^{m×n}` can be decomposed as:

$$ A = U \Sigma V^\top $$

$$ \underbrace{A}_{m \times n} = \underbrace{U}_{m \times m} \underbrace{\Sigma}_{m \times n} \underbrace{V^\top}_{n \times n} $$

|Symbol|Shape|Meaning|
|---|---|---|
|`U`|`m × m`|Left singular vectors — orthogonal matrix|
|`Σ`|`m × n`|Singular values on diagonal, zeros elsewhere|
|`Vᵀ`|`n × n`|Right singular vectors — orthogonal matrix|

---

## The Three Matrices

$$ U = \begin{bmatrix} | & | & \\ \mathbf{u}_1 & \mathbf{u}_2 & \cdots \\ | & | & \end{bmatrix}, \quad \Sigma = \begin{bmatrix} \sigma_1 & 0 & 0 \\ 0 & \sigma_2 & 0 \\ 0 & 0 & \ddots \end{bmatrix}, \quad V = \begin{bmatrix} | & | & \\ \mathbf{v}_1 & \mathbf{v}_2 & \cdots \\ | & | & \end{bmatrix} $$

- **Singular values** `σ₁ ≥ σ₂ ≥ ... ≥ 0` — always non-negative, sorted descending
- **U columns** — orthonormal basis for the output space
- **V columns** — orthonormal basis for the input space

---

## Simple Example

$$ A = \begin{bmatrix} 3 & 0 \\ 0 & 2 \end{bmatrix} = \begin{bmatrix} 1 & 0 \\ 0 & 1 \end{bmatrix} \begin{bmatrix} 3 & 0 \\ 0 & 2 \end{bmatrix} \begin{bmatrix} 1 & 0 \\ 0 & 1 \end{bmatrix}^\top $$

Here U = V = I (identity), and the singular values are just `σ₁ = 3, σ₂ = 2`.  
For a diagonal matrix, SVD is trivial — the singular values are the diagonal entries.

---

## Geometric Intuition

Any linear transformation = **rotate → scale → rotate**.

$$ A\mathbf{x} = U \Sigma V^\top \mathbf{x} $$

$$ \begin{aligned} V^\top \mathbf{x} &\quad \text{rotate/reflect input} \\ \Sigma(\cdot) &\quad \text{scale each dimension by } \sigma_i \\ U(\cdot) &\quad \text{rotate/reflect into output space} \end{aligned} $$

The singular values tell you **how much** the matrix stretches in each direction.

---

## Rank & Singular Values

$$ \text{rank}(A) = \text{number of non-zero singular values} $$

|Singular value|Meaning|
|---|---|
|Large `σᵢ`|Important direction — carries most information|
|Small `σᵢ`|Minor direction — mostly noise|
|`σᵢ = 0`|Null direction — matrix collapses this|

---

## Truncated SVD (Low-Rank Approximation)

Keep only the top `k` singular values — best rank-k approximation of A:

$$ A \approx A_k = U_k \Sigma_k V_k^\top $$

$$ \underbrace{A}_{m \times n} \approx \underbrace{U_k}_{m \times k} \underbrace{\Sigma_k}_{k \times k} \underbrace{V_k^\top}_{k \times n} $$

This is the **Eckart-Young theorem** — no rank-k matrix approximates A better (in Frobenius norm).

$$ |A - A_k|_F = \sqrt{\sigma_{k+1}^2 + \sigma_{k+2}^2 + \cdots + \sigma_r^2} $$

---

## SVD vs Eigendecomposition

|                  | Eigendecomposition                      | SVD                                |     |
| ---------------- | --------------------------------------- | ---------------------------------- | --- |
| Input            | Square matrices only                    | Any matrix                         |     |
| Output           | Eigenvalues + eigenvectors              | Singular values + singular vectors |     |
| Orthogonal?      | Only if symmetric                       | Always                             |     |
| Negative values? | Eigenvalues can be negative             | Singular values always ≥ 0         |     |
| Relation         | $\sigma_i = \sqrt{\lambda_i(A^\top A)}$ | —                                  | --- |
|                  |                                         |                                    |     |

## ML Connections

### PCA

$$ X = U \Sigma V^\top \implies \text{principal components} = V, \quad \text{variance} \propto \sigma_i^2 $$

Project data onto top-k components: `X_reduced = X Vₖ = Uₖ Σₖ`

### LoRA (Low-Rank Adaptation)

Fine-tuning insight: weight updates `ΔW` have low-rank structure.  
Instead of updating full `W ∈ ℝ^{d×d}`, learn:

$$ \Delta W = A B, \quad A \in \mathbb{R}^{d \times r},\ B \in \mathbb{R}^{r \times d}, \quad r \ll d $$

This is exactly truncated SVD applied to the update — massive parameter reduction.

### Embeddings

Word embedding matrices are approximately low-rank — SVD reveals the most meaningful semantic directions.

### Attention

The attention weight matrix `QKᵀ` is analyzed via its singular values to understand what the model is attending to.

|ML Concept|SVD Role|
|---|---|
|PCA|Truncated SVD of data matrix|
|LoRA|Low-rank approximation of weight updates|
|Compression|Reduce model size via truncated SVD|
|Noise removal|Drop small singular values|
|Spectral norm|`‖A‖₂ = σ_max`|
